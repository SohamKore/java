Java Programming Basics

Syntax

Syntax tells us how to write a specific statement in Java programming.

Statement

A statement is a line that performs an action or task. Examples:

  • int x;
  • System.out.println("Hello");

Expression

An expression is any line of code that produces a result. Examples:

  • 5 + 10
  • A + 10
  • 10 > 5

Keywords

Keywords are reserved words that are used to write a program. Examples include:

  • int, char, float, and more.

Input and Output

Methods for Output

  • System.out.print() - Prints text.
  • System.out.println() - Prints text and moves to the next line.
  • System.out.printf() - Prints formatted text.

Methods for Input

To take input in Java, follow these steps:

  1. Import the scanner class: import java.util.Scanner;
  2. Create a scanner object: Scanner sc = new Scanner(System.in);
  3. Use scanner methods to take input:
    • nextInt() – Reads an integer. Example: int num = sc.nextInt();
    • nextLine() – Reads a full line of text. Example: String line = sc.nextLine();
    • next() – Reads a single word.

Data Types

Integers

  • Byte:
    • Size: 1 byte
    • Range: -128 to 127
  • Short:
    • Size: 2 bytes
    • Range: -32,768 to 32,767
  • Int:
    • Size: 4 bytes
    • Range: -2,147,483,648 to 2,147,483,647
  • Long:
    • Size: 8 bytes
    • Range: -2^63 to 2^63

Floating Point

  • Float:
    • Size: 4 bytes
    • Range: 1.4E-324 to 1.8E+308
  • Double:
    • Size: 8 bytes
    • Range: 4.9E-324 to 1.8E+308

Other Types

  • Char:
    • Size: 1 byte
    • Range: 0 to 65,535
  • Boolean:
    • Size: 1 byte
    • Range: true or false

Operators

Arithmetic Operators

  • +, -, *, /, %

Relational Operators

  • >, <, >=, <=

Comparison Operators

  • ==, !=

Logical Operators

  • &&, ||, !

Unary Operators

  • ++, --, !

Assignment Operators

  • =, +=, -=
